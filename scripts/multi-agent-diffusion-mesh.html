Multi-agent mesh + diffusion steering

6400 agents on an 80×80 grid. RGB from each DDPM step decodes to a displacement vector per vertex (fixed-linear [−15,15] per channel, same as training). Cohesion, separation, and alignment are local 8-neighbor boids; diffusion pulls each agent toward the decoded surface target.
